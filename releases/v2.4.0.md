- [FEATURE] Added AI-powered summarization for daily reports using Gemini 1.5 Flash.
  > Users can now click the 'Summarize' button on any report > 1000 words to get a bulleted summary. This features leverages the new `gemini-1.5-flash` model for sub-second latency.

- [IMPROVEMENT] Reduced dashboard initial load time by 40%.
  > Optimized React bundle size and implemented aggressive code-splitting for heavy charting libraries.

- [BUGFIX] Fixed an issue where dark mode preferences were not persisting across sessions.
